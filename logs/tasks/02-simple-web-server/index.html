Для начала взаимодействие фронта и бэка будет очень простым, думаю, что достаточно будет отдавать на фронт статичные html-страницы. Для этого нужно в Smalltalk поднять простой веб-сервер. Можно обойтись стандартными средствами Zinc Http, а можно использовать очень тонкую прослойку Teapot. Ее в образе нет, нужно будет загрузить и подключить как зависимость в проект:

Metacello new
  baseline: 'Teapot';
  repository: 'github://zeroflag/Teapot/source';
  load.

Использовать потом должно быть очень просто, документация по библиотеке доступна здесь.
