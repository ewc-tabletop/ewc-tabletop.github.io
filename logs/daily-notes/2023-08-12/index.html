Вчерашний стрим показал, что скорости интернет-соединения явно недостаточно для "живых" сессий. Поэтому перехожу на режим оффлайн-записи скринкастов и их последующей публикации на youtube-канале. Лайвкодинг сам по себе мне не важен, разве что наличие в календаре стрима - это обязательство, не дающее "слиться" в процессе. Ну что тут сказать... поживем - увидим.

Сегодня создал в пространстве организации на github два новых репозитория: один со статичным сайтом, предназначенным для публикации на github pages; второй - с исходниками для этой самой публикации. Выбрал mkdocs в качестве генератора статичных сайтов, потому что он простой и закрывает все мои текущие потребности в ведении документации / дневника по разработке.
